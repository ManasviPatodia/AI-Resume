# AI Resume Builder

AI Resume Builder is a full-stack web application that helps users create, edit, and manage professional resumes. It integrates Google Gemini AI to generate and improve resume content while providing ATS-friendly suggestions and customizable templates.

## Features

- User authentication
- Create and manage multiple resumes
- AI-powered resume content generation
- ATS-friendly optimization
- Real-time resume preview
- Public resume sharing
- Image upload with ImageKit

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Gemini AI
- ImageKit
